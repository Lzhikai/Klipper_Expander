# Indicator light    
*    ## [D1] Connection status light    
*    ## [D2-D5] Switch on indicator    
*    ## [D6] 5V status light    
*    ## [D7] VINSupply status

![](/images/1.png)    
### 2.You need to add the configuration of the inside the `printer.cfg` file.    
```Bash
[include klipperExpander.cfg]
```
